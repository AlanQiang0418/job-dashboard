# 求职申请管理看板

这是一个面向大学生求职季场景的单页网页原型，适合作为产品设计测评题的演示作品。

## 作品亮点

- 用看板视图管理申请阶段：待投递、已投递、笔试/面试、Offer/结果
- 用任务中心聚焦当前需要处理的岗位，按轻重缓急排序
- 用表格管理查看申请详情，并支持搜索、筛选和排序
- 用材料中心统一管理简历、作品集、成绩单等求职材料
- 用面试日程和推荐岗位补充求职过程中的关键动作入口
- 支持新增申请、编辑详情、导出周报、导出示例日历文件
- 支持 `localStorage` 本地持久化，刷新页面后申请记录不会丢失
- 支持一键恢复默认示例数据，便于演示后重置

## 使用方式

直接打开 `index.html` 即可预览。

说明：本地持久化基于浏览器 `localStorage`，部署到 GitHub Pages 后同样可用，但只在当前浏览器和当前设备内保留。

## 文件说明

- `index.html`：页面结构
- `styles.css`：页面样式
- `script.js`：交互逻辑与示例数据
